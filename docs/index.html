在网页世界里
垂钓你的数据

垂钓鸭是一款面向网页数据采集与自动化的桌面应用。通过 Chrome 插件控制浏览器，无需调试模式，选技能包、建任务、自动执行、导出结果，一条链路搞定。

⬇ 下载 Windows 版 查看源码

核心流程

1安装 Chrome 插件并配对
2选择技能包，创建任务
3调度中心监控执行状态
4导出 Excel / JSON 结果

产品特性

为真实网页场景设计，轻量、可控、可扩展

🧩

Chrome 插件驱动

正常打开 Chrome 即可，无需调试端口，Cookie 与登录态完整保留。

🎣

技能包架构

每个技能包内置完整流程与默认参数，选包命名即可运行，支持扩展。

📋

任务与调度

任务中心管理方案与历史，调度中心聚焦运行中与报错，支持暂停与重试。

💾

本地数据

SQLite 存储执行记录，流程与日志可追溯，数据留在本机。

📊

多格式导出

支持 Excel、JSON、CSV 等格式，统一在导出中心管理。

⚡

Electron 桌面端

Vue 3 + TypeScript 构建，界面现代，Windows 开箱即用。

快速上手

首次配置约 2 分钟，之后可反复使用

Step 1

安装插件

在浏览器管理中打开插件目录，加载到 Chrome 并填写配对码。

Step 2

选择 Tab

在 Chrome 打开目标网页，回到应用选择要操作的标签页。

Step 3

新建任务

选择技能包、填写任务名称，一键启动自动化流程。

Step 4

查看结果

在调度中心跟踪进度，在导出中心获取采集文件。

下载桌面版

目前提供 Windows x64 绿色压缩包，解压后运行 ChuiDuck.exe 即可。

需要 Windows 10 及以上 · 需安装 Google Chrome
